# Geoloc

## Dependencias código Haskell

* Instalar paquetes con `cabal` (paquete `cabal-install` en `dnf`) así:

    ```bash
    $ cabal update
    $ cabal install <paquete>
    ```

* Paquete `munkres` para método húngaro, agradecimientos a *Balazs Komuves*.

## Dependencias documento
